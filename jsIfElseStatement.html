JavaScript If Else Statement

In thie lesson, we will study about conditional statements.
Conditional statements are used to execute code based on different conditions.

The if Statement

The if statement executes a block of codes if the given condition is true.

Syntax:

if(condition) {
// block of codes to be executed when the condition is true
}

In the example below, if age is more than 18 a pop-up dialog will be shown.

Sample code:

if(age > 18) {
// alert("User is old enough to enter");
}

The if...else Statement

The else statement executes a block of codes when the given condition to "if" is false.

Syntax:

if(condition) {
// block of codes to be executed when the condition is true
} else {
// block of codes to be executed when the condition is false
}

Here is an example:

var age = 15;
if(age > 18) {
// alert("user is old enough to enter");
} else {
// alert("user is NOT old enough to enter");
}

Here's an illustrative example, the user will be asked to enter their age.

If they are above 18, they will be allowed to enter.
If they are below 18, they will NOT be allowed to enter.


document.getElementById("submit").addEventListener("click", function(){
var age = parseInt(document.getElementById("age").value);
if(age >= 18) {
alert("user is old enough to enter");
} else {
alert("user is NOT old enough to enter");
}
});
