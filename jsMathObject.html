JavaScript Math Object

The Math object, is buit-in JavaScript object.
The Math object contains properties and methods that allow us to do mathematical tasks.

The ceil() Method

Tha Math.ceil() method rounds up a number to the nearest whole number or integer.

var num1 = Math.ceil(.90);
var num2 = Math.ceil(1.70);
var num3 = Math.ceil(21.60);
document.getElementById("demo").innerHTML =
(num1) + ", " + (num2) + ", " + (num3);

The Math Properties

The Math object contains the following properties.

Property Description Approximate Value
Math.E Represents the base of natural logarithms. 2.718
Math.LN10 Represents the natural logarithm of 10. 2.303
Math.LN2 Represents the natural logarithm of 2. 0.693
Math.LOG10E Represents the base 10 logarithm of e. 0.434
Math.LOG2E Represents the base 2 logarithm of e. 1.443
Math.PI Represents the ratio of the circumference ot a circle to its diameter. 3.14159
Math.SQRT1_2 Represents the square root of 1/2. 0.707
Math.SQRT2 Represents the square root of 2. 1.414
