What is an Object

An object is a collection of related data and/or functionality.
Most of the time, creating an object begins with defining a variable.

// sample object this will not print on the browers
var person = {
fullname: "Hi Hello",
age: 22,
hobbies: ["coding","playing chess", "gaming"],
isMarried: true
};

As you can see, the syntax of an object is pretty simple.
The person object contains properties.
Each property has a name (e.g fullname, age) and a value (e.g "Hi Hello", 30).
Properties are separated by commas ( , ).
The name and value pair are separated by a collon ( : ).

Methods in an Object

An object can also contain functions (methods).

var person = {
fullname : "Hi Hello",
age : 22,
showName : function(){
alert(this.fullname);
},
showAge : function() {
alert(this.age);
}
};

Note! an object is a collection of properties and/or methods.

Accessing Object's Properties

There are two ways to access an object's members (properties and methods).

The Dot Notation

The Dot Notation uses to a dot ( . ) to access an object's members.
First write the variable, then the name fo the property method.

The Bracket Notation

We can also use square brackets to access an object's members.
As you may notice, the brackets contain the names of the properties.

The this Keyword

You may have notice the this keyword in one of our examples. This may look strange, but don't worry. It's easy.
The this keyword refers to the current object.
