JavaScript For Loop

When programming, there are times that we need to run the same statement over and over again.
In this example, we accessed each value of fruits array one-by-one.

var fruits = ["apple","banana","mango","orange","strawberry"];
var html = "";
html += "<p>" + fruits[0] + "</p>";
html += "<p>" + fruits[1] + "</p>";
html += "<p>" + fruits[2] + "</p>";
html += "<p>" + fruits[3] + "</p>";
html += "<p>" + fruits[4] + "</p>";
document.getElementById("demo").innerHTML = html;

Sample output:

The code above can be shortened using the for loop.

var fruits = ["apple","banana","mango","orange","strawberry"];
var html = "";
for(var i = 0; i < fruits.length; i++) {
html += "<p>" + fruits[i] + "</p>";
}
document.getElementById("demo1").innerHTML = html;

Sample output:


Notice that both examples above work the same way, but with different approaches.

What is for loop ?

A for loop executes statements repeatedly as long as the sprecified condition is met.

Syntax:

for(initialExpression; condition; incrementExepression) {
// statements or code blocks
}

  • initialExpression - this intializes the loop counter
  • condition - if it is true, the loop statments execute. if false, the stements don't execute.
  • incrementExepression - increments the loop counter, runs everytime the statements are executed
The statement below will run 5 times.

for(var i = 0; i < 5; i++) {
document.write("<p> Hello World! </p>");
}

Sample output:
In the example above, the loop initialized with i = 0.
Next, the condition is define, and i must be less than 5 for the loop to continue to run.
Then, the i++ increments the counter everytime the code blocks are executed.
Once i is NO LONGER less than 5, the loop stops.
The for loop is commonly used with arrays.

var cities = ["Mumbai", "Abuja", "New York"];
var html = "";
for(var i = 0; i < cities.length; i++){
