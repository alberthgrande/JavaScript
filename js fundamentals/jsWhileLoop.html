JavaScript Break and Continue

The while loop execution statements as long as th given condition evaluates to true.

Syantax:

while(condition){
// statements or code blocks
}

  • condition - the expression evaluated,
    if it evaluates to true, the code blocks are
    executed. if false,the loops stops
  • statements - the code blocks to be
    executed as long as the condition is true
Here is a basic example:

var i = 0;
while(i < 3) {
document.write("<p> Hello World! </p>");
i++;
}

Sample output:
In the example above the statement is executed 3 times.
It is because the statement only stopped running when i is NO LONGER less than 3.

Looping Over an Array Using while Loop

To loop over an array, declare a variable that holds the length of the array.
Then use it in the condition.

var fruits = ["apples","bananas","oranges"];
var len = fruits.length;
var html = "";
var i = 0;
while(i < len) {
html+= "<p>" + fruits[i] + "</p>";
i++;
}
document.getElementById("demo").innerHTML = html;

The do...while Loop

The do...while loop executes statements until the test condition evaluates false.

Syntax:

do {
// statements or code blocks
}while(condition);

  • statements - the code block executed and is re-executed each time the condition is true
  • condition - the expression evaluated, if it evaluates to true, the code blocks re-executed. If false, the loop stops.
Here is a basic example:

var i = 0;
do {
document.write("<p> Hello World! </p>");
i++;
}while(i < 3);

Sample output:
In the example above the statement is executed 3 times.
It is because the statement only stopped running when i is NO LONGER less than 3.

Looping Over an Array Using do...while Loop

We can also use do...while loop to loop over an array.

var cities = ["Berlin","Tokyo","Rio","Denver","Stockholm"];
var cityLen = cities.length;
var html = "";
var i = 0;
do{
html += "<p> Someday, I want to go to " + cities[i] + "</p>";
i++;
}while(i < cityLen);
document.write(html);
