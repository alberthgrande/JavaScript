JavaScript Strings

In JavaScript, strings are pieces of the text.
A string is set of characters enclosed in qoutes.

Creating Strings

Strings can be enclosed in double qoutes ( "" ) .
Strings can be enclosed in signle qoutes ( '' ) .

Single or Double qoutes

Use single qoutes if your string contains double qoutes.
Use double qoutes if your string contains single qoutes.

If you do it like the example below, your code will result to an error because it will confuse the browser.

var str = 'Everything's fine;
document.write(str);

Bad String

A bad string is a string that is not surrounded by the qoutes.

var badString1 = I am a text;
var badString1 = "I am a text;
var badString1 = I am a text";

You should avoid these, because it will produce an error.

Escaping Characters

Escaping characters is crucial in handling strings.
It helps us make sure that our strings are recognized as pieces of the text, not as part of the code.
To escape characters, use a backslash ( \ ) right before the character.
