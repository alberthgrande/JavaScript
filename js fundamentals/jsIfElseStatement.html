JavaScript If Else Statement

In thie lesson, we will study about conditional statements.
Conditional statements are used to execute code based on different conditions.

The if Statement

The if statement executes a block of codes if the given condition is true.

Syntax:

if(condition) {
// block of codes to be executed when the condition is true
}

In the example below, if age is more than 18 a pop-up dialog will be shown.

Sample code:

if(age > 18) {
// alert("User is old enough to enter");
}

The if...else Statement

The else statement executes a block of codes when the given condition to "if" is false.

Syntax:

if(condition) {
// block of codes to be executed when the condition is true
} else {
// block of codes to be executed when the condition is false
}

Here is an example:

var age = 15;
if(age > 18) {
// alert("user is old enough to enter");
} else {
// alert("user is NOT old enough to enter");
}

Here's an illustrative example, the user will be asked to enter their age.

If they are above 18, they will be allowed to enter.
If they are below 18, they will NOT be allowed to enter.


document.getElementById("submit").addEventListener("click", function(){
var age = parseInt(document.getElementById("age").value);
if(age >= 18) {
alert("user is old enough to enter");
} else {
alert("user is NOT old enough to enter");
}
});

The else if Statement

The else if statement executes a block of codes if its specified condition is true and the first condition is false.

Syntax:

if(condition) {
// block of codes to be executed when the condition1 is true
} else if(condition) {
// block of codes to be executed when the condition2 is false
} else {
// block of codes to be executed when the condition1 and condition2 are false
}

Here is a basic example:

var x = 15;
if(x < 10) {
document.getElementById("demo").innerHTML = "x is less than 10";
} else if(x > 10) {
document.getElementById("demo").innerHTML = "x is more than 10";
} else {
document.getElementById("demo").innerHTML = "x is 10 because it's neither less than or more than 10";
}

The example below is greets the user depending on the current time of the day.

If the time is below 11:00, it says "Good morning."

If the time is below 18:00 but more than 11:00, it says "Good day."

Otherwise "Good evening."

function greetUser(){
var time = new Date().getHours();
if(time < 11) {
return "Good morning";
} else if(time < 18) {
return "Good day";
} else {
return "Good evening";
}
}
document.getElementById("demo1").innerHTML = greetUser();
