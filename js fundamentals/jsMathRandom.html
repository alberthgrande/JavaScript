JavaScript Math Random

In this lesson, you will learn how to generate a random number.

The random() Method

The Math.random() returns a randon floaring-point (number with decimals) between 0 (include) and 1 (exclude).
To get the random number refresh the page.

var random = Math.random();
document.getElementById("demo").innerHTML = random;

Get Random Integer Between Two Numbers

The example above only gets a random floating-point (number with decimals) between 0 (inclusive) and 1 (exclusive).

What if we need to get a random integer between two specified integers ? Here is how.

function getRandomInt(min, max) {
document.getElementById("demo1").innerHTML = Math.floor(Math.random() * (max - min) + min);
}

Sample output:



To get a random number that also includes the maximum number, use this function.

function getRandomInt(min, max) {
document.getElementById("demo2").innerHTML = Math.floor(Math.random() * (max - min + 1) + min);
}
