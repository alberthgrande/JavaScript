JavaScript Fucntions

Fucntions are one of the fundamental building blocks of JavaScript.
A fucntion is a JavaScript Method
It is a set of statements that performs a taks or produces a value.
It is executed when it is invoked (called) by something.

Fucntion Definitions

A JavaScript fucntiion is defined or declared using the fucntion keyword, followed by:

  • The name of the function.
  • A list of paramenters (optional) enclosed in parentheses ( ). Multiple parameters are separeated by commas ( , ).
  • The block of the codes or statements enclosed in curly bracess { }.
Parameters act as placeholders inside the function.
When the fucntion is called, the variables are assigned to the arguments provided when the fucntion is called.

Calling a Fucntion

Function will only be executed when they are called or invoked.
A Fucntion can be called inside a JavaScript program, just like what we did in the previous examples
A fucntion can be called when an event happens, most commonly when a button is clicked.
