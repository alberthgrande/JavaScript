JavaScript for..of Loop

The for..of statement loops over iterable object. The most basic iterable object are arrays and strings.

Syntax:

for(variable of iterable) {
// statement or code blocks
}

  • variable - on each iteration the value of the property is assigned to variable
  • iterable - the iterable object being iterated over
Here is an example of looping through the values of an array:

var cities = ["Mumbai","Abuja","Manila"];
for(value of cities) {
document.write(value + "<br>");
}

var text = "codeliber";
for(letter of text) {
document.write(letter + "<br>");
