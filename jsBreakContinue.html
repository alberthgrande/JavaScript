JavaScript Break and Continue

The break and continue statements are often in loops.

The break Statement

You have already learned that the break statement is used to stop the execution of the switch statement.
In this lesson, you will learn how to use in loops.
The break statement terminates the execution of the current loop.
After termination, the execution proceeds to the next statement following the loop.

var html = "";
for(var i = 0; i < 10; i ++) {
if(i == 5) {
break;
} html += "<li> Number " + i + "</li>";
}
document.getElementById("demo").innerHTML = html;

Sample output:

As you may noticem instead of printing 10 numbers, only 5 numbers were printed.
It is because when i became equal to 5 the next iteration was stopped using the break statement.
After that, the program executed the following statement that printed the html string, instead of adding more strings to the html variable.
And the continues the execution of the loop with the next iteration.
var html = "";
for(var i = 0; i < 10; i++) {
if(i == 5) {
continue;
}
html += "<li> Number " + i + "</ li>";
}
document.getElementById("demo1").innerHTML = html;

Sample output:

As you may notice, in the example above, only "Number 5" was missing.
It is because the continue statement terminated the execution of the current iteration when i became equal to 5.
After that, the program continued the execution of the next iteration.
