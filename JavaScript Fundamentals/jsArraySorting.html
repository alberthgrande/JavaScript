JavaScript Array Sorting

In this lesson, you will learn how to sort arrays using Array method.

The sort() Method

The sort() method sorts the elements of an array.
Assending is the default sort order.
In the example below, the array sorted alphabetically.

var fruits = ["Banana", "Apple", "Durian", "Cherries"];
fruits.sort();
document.write(fruits);

The Compare Function

The sort() method can take one argument, a compare function.

The compare function defines the sort order.
If it ommitted (like in the example above), array elements are converted to string the sorted according to each character's Unicode code point value.
The compareFunction() comes with two argument a and b.
If a and b are two elements being compared then:

  • If compareFunction(a, b) returns less than 0, a comes first.
  • If compareFunction(a, b) returns 0, a and b are unchange.
  • If compareFunction(a, b) returns greater than 0, b comes first.
The examples below use compare function.

Sorting Numbers

To sort numbers in an array, we should us a compare function.
In the example below, the numbers are sorted in asscending order.

var numbers = [5, 3, 1, 2, 4];
numbers.sort(function(a, b){
return a - b;
});
document.write(numbers);

Get the Highest or Lowest Value

To get the highest value in an array of numbers, sort the element in descending order than get the first value.

var numbers = [5, 3, 1, 2, 4];
numbers.sort(function(a, b){
return b - a;
});
var highest = numbers[0]; // get the first value
document.write(highest);

Toget the lowest value in an array of numbers, sort the elements in ascending order then get the first value.

var numbers = [5, 3, 1, 2, 4];
numbers.sort(function(a, b) {
return a - b;
}); var lowest = numbers[0]; // get the first value
document.write(lowest);

Sorting an Array of Objects

To sort an array of object, simply coampare the object's numerical properties.

var students = [
{name: "John", age: 7},
{name: "Monica", age: 5},
{name: "Paul", age: 6}
];
students.sort(function(a, b) {
return a.age - b.age; });
/* displaying data */
for(var i = 0; i < students.length; i++){
document.write("Name: " + students[i].name + " Age: " + students[i].age);
}

Reversing an Array

The reverse() method reverses an array.

var fruits = ["Apples","Bananas","Oranges"];
fruits.reverse();
document.write(fruits);
