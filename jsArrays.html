Create an Array

Arrays are created with square brackets [] and data are separated with commas, ( , ).

var cities = ["Madrid", "Berlin", "Tokyo"];

Sample Output:

Another way of creating an array is the Array() constructor.
var cities = new Array("Madrid", "Berlin", "Tokyo");

Sample Output:

It is recommended to use square brackets it's much faster in execution and more readable.

Access an Array Item

Array items are accessed using interger.
Arrays are zero-indexed. This means the count starts from 0, not from 1.

var cities = ["Madrid","Berlin","Tokyo"];
document.getElementById("demo5").innerHTML = cities[0];

Sample Output:

Notice that cities[0] returned the first element, which is "Madrid".

Add Items to an Array

The push() method adds one or more elements (items) to the end of an array.
Here's and example of adding a single element:
var cities = ["Madrid","Berlin","Tokyo"];
cities.push("Rio");

Sample Output:



The unshift() method adds one or more elements (items) to the front of an array.
Here's an example of adding a single element:
var ceties = ["Madrid","Berlin","Tokyo"];
