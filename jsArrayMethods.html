JavaScript Array Methods

In this lesson, we'll study the most useful Array methods.

The forEach() Method

The forEach() method is another way to loop over an array, or acces each element's data.
It executes a callback function for each array element.

var cities = ["Madrid","Berlin","Tokyo"];
cities.forEach(function(val, i, arr){
document.write(i + ": " + val + );
});

Sample Output:
The callback function is invoked with 3 arguments.

  1. The value of the element.
  2. The index of the element.
  3. The array itself.

Arrays to Strings

The join() method creates and returns a new string by combining all the elements.
The elements ate separated by the specified separator charater.
If none is specified, it separates the elements with commas.

var cities = ["Madrid","Berlin","Tokyo"];
var str = cities.join();
document.write(str);

Sample Output: Here's is an example with specified separator:
var cities = ["Madrid","Berlin","Tokyo"];
var str = cities.join("-");
document.write(str);

Sample Output: Note! The join() method does not modify the existing arrays, instead it create and returns a new one.

Strings to Arrays

It is also possible to convert a string into a array, this can be done using the split() method.
The split() method takes a separator string as its argument.
The method splits the string at each point where the separator occurs.

var string = "Lorem ipsum dolor sit amet.";
var arr = string.split(" ");
document.write(typeof arr + ": " + arr);

Sample Output: Note! The split() method does not modify the existing string, It creates a new array instead.

Here's another example:
var string = "Lorem ipsum dolor sit amet.";
var arr = string.split("-");
document.write(typeof arr + ": " + arr);

Sample Output: Remember! The data type of arrays is object.

Merging Arrays

The concat() method merges two or more arrays.

var fruits1 = ["Banana","Apple","Orange"];
var fruits2 = ["Grape","Mango","Strawberry"];
var arr = fruits1.concat(fruits2);
document.write(arr);

Sample Output: To concatenate three or more arrays, separate the arrays with commas.

var numbers1 = [1,2,3];
var numbers2 = [4,5,6];
var numbers3 = [7,8,9]; // this part is to concatenate
var arr = numbers1.concat(numbers2,numbers3);
document.write(arr);

Note! The concat() method does not modify the existing arrays, instead is creates and return a new one.

Searching an Item

The indexOf() method returns the index of the first occurence of an Item (element) in the array.

var fruits = ["Mango","Apple","Orange","Grape","Mango"];
var search = fruits.indexOf("Mango"); // returns 0
document.write(search);

Sample Output:

The lastIndexOf() method returns the index of the last occurence of an item (element) in the array.

var fruits = ["Mango","Apple","Orange","Grape","Mango"];
var search = fruits.lastIndexOf("Mango");
document.write(search);

Sample Output:

Rememeber! 0 is the index of the first element, 1 is the second element, and so on.
Both the indexOf() and lastIndexOf() methods return -1 if the item is not found in the array.

var fruits = ["Mango","Apple","Orange","Grape","Mango"];
var firstIndex = fruits.indexOf("Strawberry"); // returns -1
var lastIndex = fruits.lastIndexOf("Strawberry"); // returns -1
