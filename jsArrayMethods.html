JavaScript Array Methods

In this lesson, we'll study the most useful Array methods.

The forEach() Method

The forEach() method is another way to loop over an array, or acces each element's data.
It executes a callback function for each array element.

var cities = ["Madrid","Berlin","Tokyo"];
cities.forEach(function(val, i, arr){
document.write(i + ": " + val + );
});

Sample Output:
The callback function is invoked with 3 arguments.

  1. The value of the element.
  2. The index of the element.
  3. The array itself.

Arrays to Strings

The join() method creates and returns a new string by combining all the elements.
The elements ate separated by the specified separator charater.
If none is specified, it separates the elements with commas.

var cities = ["Madrid","Berlin","Tokyo"];
var str = cities.join();
document.write(str);

Sample Output: Here's is an example with specified separator:
var cities = ["Madrid","Berlin","Tokyo"];
var str = cities.join("-");
document.write(str);

Sample Output: Note! The join() method does not modify the existing arrays, instead it create and returns a new one.

Strings to Arrays

It is also possible to convert a string into a array, this can be done using the split() method.
The split() method takes a separator string as its argument.
The method splits the string at each point where the separator occurs.

var string = "Lorem ipsum dolor sit amet.";
var arr = string.split(" ");
document.write(typeof arr + ": " + arr);

Sample Output: Note! The split() method does not modify the existing string, It creates a new array instead.

Here's another example:
var string = "Lorem ipsum dolor sit amet.";
var arr = string.split("-");
document.write(typeof arr + ": " + arr);

Sample Output: Remember! The data type of arrays is object.

Merging Arrays

The concat() method merges two or more arrays.

var fruits1 = ["Banana","Apple","Orange"];
var fruits2 = ["Grape","Mango","Strawberry"];
var arr = fruits1.concat(fruits2);
document.write(arr);

Sample Output: To concatenate three or more arrays, separate the arrays with commas.

var numbers1 = [1,2,3];
var numbers2 = [4,5,6];
var numbers3 = [7,8,9]; // this part is to concatenate
var arr = numbers1.concat(numbers2,numbers3);
document.write(arr);

Sample Output:

Note! The concat() method does not modify the existing arrays, instead it creates and return a new one.

Searching an Item

The indexOf() method returns the index of the first occurence of an Item (element) in the array.

var fruits = ["Mango","Apple","Orange","Grape","Mango"];
var search = fruits.indexOf("Mango"); // returns 0
document.write(search);

Sample Output:

The lastIndexOf() method returns the index of the last occurence of an item (element) in the array.

var fruits = ["Mango","Apple","Orange","Grape","Mango"];
var search = fruits.lastIndexOf("Mango");
document.write(search);

Sample Output:

Rememeber! 0 is the index of the first element, 1 is the second element, and so on.
Both the indexOf() and lastIndexOf() methods return -1 if the item is not found in the array.

var fruits = ["Mango","Apple","Orange","Grape","Mango"];
var firstIndex = fruits.indexOf("Strawberry"); // returns -1
var lastIndex = fruits.lastIndexOf("Strawberry"); // returns -1
document.write(firstIndex , lastIndex);

Sample Output:


To check if a value exist in an array, use the includes() method.
The include() method returns true if a certain value is include in an array.

var cities = ["Madrid","Berlin","Tokyo"];
var city1 = cities.includes("Madrid"); // return true
var city2 = cities.includes("Manila"); // return false
document.write("City1: " + city1 + ", City2: " + city2 );

Slicing an Array

The slice() method returns a new copy of a section of an array.
It takes two arguments:

  1. begin - the index where the extraction begins
  2. end - the index where the extraction should end (it is not include in the new array)
var cities = ["Madrid","Berlin","Tokyo","Rio","Denver"];
var arr = cities.slice(0, 2); // return "Madrid", "Berlin"
document.write(arr);

Sample Output:

Here's is another example:

var cities = ["Madrid","Berlin","Tokyo","Rio","Denver"];
var arr = cities.slice(1, 4); // returns "Berlin","Tokyo","Rio"
document.write(arr);

Sample output:

Note! The slice() method does not modify the existing arrays, instead it creates and return a new one.

Splicing an Arrays

The splice() method changes the contents of an array by removing or replacing existing elements and/or adding new elements in place.

It takes three or more arguments:

  1. start - the index where the changing begins
  2. deleteCount - the number of the elements to remove and replace from the start
  3. element1, element2, ... - the elements to add to the array
var months = ["January","April","May"];
months.splice(1, 0, "February", "March");
document.write(months);

Sample output:

Removing an Array item

The splice() method can also be used to remove an array item.
To remove item/s, omit the third argument.

To remove a single item, specify the index of the item you want to remove and write 1 as the value of the second argument.

var months = ["January","February","March","April","May"];
months.splice(0, 1); // returns "February","March","April","May"
document.write(months);

Sample output:

Here's is another examples:

var months = ["January","February","March","April","May"];
months.splice(2, 1); // returns "January","February","April","May"

To remove multiple items, specify the index where you want to start removing the firt argunebt, then specify the number of the items tou want to remove in the second argument.

document.write(months);
