What is an Object

An object is a collection of related data and/or functionality.
Most of the time, creating an object begins with defining a variable.

// sample object this will not print on the browers
var person = {
fullname: "Hi Hello",
age: 22,
hobbies: ["coding","playing chess", "gaming"],
isMarried: true
}

As you can see, the syntax of an object is pretty simple.
The person object contains properties.
Each property has a name (e.g fullname, age) and a value (e.g "Hi Hello", 30).
Properties are separated by commas ( , ).
The name and value pair are separated by a collon ( : ).

Methods in an Object

An object can also contain functions (methods).

var person = {
fullname : "Hi Hello",
age : 22,
showName : function(){
alert(this.fullname);
},
showAge : function() {
alert(this.age);
}
}
